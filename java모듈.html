모듈이란?

관련된 변수, 함수, 클래스 등을 패키지화하여 다른 프로그램에서 재사용할 수 있도록 합니다.

순수한 자바스크립트에서는 모듈이라는 개념이 존재하지않지만 구동되는 호스트환경에 따라서 서로다른 모듈화 방법이 제공되고있다

사용하는 이유

  • 자주 사용되는 코드를 별도의 파일로 만들어서 필요할 때마다 재활용할 수 있다.
  • 코드를 개선하면 이를 사용하고 있는 모든 애플리케이션의 동작이 개선된다.
  • 코드 수정 시에 필요한 로직을 빠르게 찾을 수 있다.
  • 필요한 로직만을 로드해서 메모리의 낭비를 줄일 수 있다.
  • 한번 다운로드된 모듈은 웹브라우저에 의해서 저장되기 때문에 동일한 로직을 로드 할 때 시간과 네트워크 트래픽을 절약 할 수 있다. (브라우저에서만 해당)

모듈의 사용

html내에잇는 자바스크립트 부분을 ~.js라는 파일을 만들어 따로 놓을수있다


모듈의 종류

  • Node.js : 코드 구조화와 재사용성을 높여줍니다
  • 라이브러리 : 라이브러리는 모듈과 비슷한 개념이다.라이브러리는 자주 사용되는 로직을 재사용하기 편리하도록 잘 정리한 일련의 코드들의 집합이다
  • jQuery : 다양한 웹 개발 작업을 간소화하고 효율적으로 처리할 수 있는 도구입니다
