배열

배열이란?

연관된 데이터를 모아서 통으로 관리하기 위해서 사용하는 데이터 타입이다 변수가 하나의 데이터를 저장하기위한 것이라면 배열은 여러개의 데이터를 저장하기 위한 것이다

배열의 생성

배열은 []대괄호를 사용하여 만들수있다 대괄호안에 데이터는 ,로 구분해서 나열하면 배열이된다

var member = ['egoing', 'k8805', 'sorialgi']

그리고 각각의 데이터를 원소라고 부른다 이것을 꺼내기위해선 대괄호안에 숫자를(색인)넣으면 된다

var member = ['egoing', 'k8805', 'sorialgi']
alert(member[0]);
alert(member[1]);
alert(member[2]);

배열이 없을때

예를 들어 멤버의 이름을 제공하는 함수를 제공해야한다 할때 함수의 하나의 값만 반환할수있다

function get_member1(){
return 'egoing';
}
document.write(get_member1());

function get_member2(){
return 'k8805';
}
document.write(get_member2());


function get_member3(){
return 'sorialgi'
}
document.write(get_member3());

위와 같이 회원정보를 반환하는 함수를 만들었는데 뭔가 너무 복잡해보이고 어려워보인다 이건 배열을통해 해결할수잇다

function get_members(){
return ['egoing', 'k8805', 'sorialgi']; }
var members = get_members();

document.write(members[0]);
document.write(members[1]);
document.write(members[2]);

함수를 만들고 반환하는것이다.


배열과 반복문 응용

배열은 반복문을 통해서 인자들을 하나하나 계산하는것이 가능하다.

반복문을 이용해서 배열 members의 내용을 하나씩 꺼낸 후에 이름의 첫글자를 대문자로 변경한 후에 출력하고 있다. 정리하면, 배열이란 연관된 정보를 하나의 그룹으로 관리하기 위해서 사용한다. 그리고 그 정보를 처리 할 때는 반복문을 이용한다.



배열을 제어시키는 데이터 타입들

  • 배열의 크기 :var arr = [1, 2, 3, 4, 5]; alert(arr.length); length를 사용하여 배열의 크기를 알수있다

  • 배열의 조작 : li.push('f'); 이렇게 사용이 가능하며 push로 배열에 원소를 추가할수잇다

  • 배열의 조작 2 : li = li.concat(['f', 'g']); 이렇게 사용이 가능하며 concat을 사용하여 추가할수잇다.(복수의 원소)

  • 배열의 조작 3 : li.unshift('z'); 이렇개 사용할수있으며 추가하는 방식이다(배열의 시작점)

  • 배열의 조작 4 : li.splice(2, 0, 'B'); 이렇게 사용할수있으며 추가하는 방식이다(자리수를 정하고 그 자리수에 추가함)

  • 배열의 제거 : li.shift();이렇게 사용할수있으며 제거하는방식이다

  • 배열의 제거2 : li.pop(); 이렇게 사용할수있으며 제일뒤에 원소를 제거하는방식이다

  • 배열의 정렬 : li.sort(); 이렇게 사용할수있으며 정렬하는 방식이다

  • 배열의 정렬 : li.reverse(); 이렇게 사용할수있으며 역순으로 정렬하는 방식이다
