메서드와 this편

매세드란 객체에 속한 함수를 가르키는 용업입니다

메서드 만드는법은

let user = {
name: "John",
age: 30
};
user.sayHi = function() {
alert("안녕하세요!");
};
user.sayHi(); // 안녕하세요!


위코드에서 user에 할당된 sayhi가 즉 객체 프로퍼티에 할당된 함수니 이게 메서드가 됩니다

객체지향 프로그래밍이란 객체를 사용하여 개체를 표현하는 방식을 말합니다

메서드와 this

this는 객체를 나타냅니다(메서드 호출 객체)

메서드는 객체에 정보에 접근할수 잇어야합니다

let user = {
name: "John",
age: 30,
sayHi() {
alert(user.name); // 'this' 대신 'user'를 이용함
}
};

위 코드에서 this는 user를 나타냅니다 물론 this도 안쓰고 다가갈수잇지만 오류가 생길수도있습니다

this의 특별함

this는 자유롭습니다 그이유로는 자바스크립트의 this는 다른 프로그래밍 언어와 다르기 때문인데요

자바스크립트에선 모든 함수에 사용할수있습니다

function sayHi() {
alert( this.name );
}
코드를 보면 메서드가 없지만 사용된 모습입니다

그리고 객체도 없이 사용할수있습니다

function sayHi() {
alert(this);
}
sayHi(); // undefined
