객체편

객체의 만드는법

let user = new Object(); // '객체 생성자' 문법
let user = {}; // '객체 리터럴' 문법

객체 안은 키와 값의 쌍으로 구성된 프로퍼티가 들어갑니다

let user = { // 객체
name: "John", // 키: "name", 값: "John"
age: 30 // 키: "age", 값: 30
};

콜론을 기준으로 왼쪽은 키 오른쪽은 값이 들어갑니다

이 프로퍼티를 추가 삭제할수도잇습니다 (읽기도 가능)

    프로퍼티 값얻기
  • alert( user.name ); // John
  • alert( user.age ); // 30
    삭제
  • delete user.age;

여기서 프로퍼티를 묶을땐 이름을따움표로 묶어줘야합니다 끝은 쉼표로 끝낼수있습니다

계산된 프로퍼티란 프로퍼티키가 대괄호로 둘러쌓인 경우를 말합니다

let fruit = prompt("어떤 과일을 구매하시겠습니까?", "apple"); let bag = { [fruit]: 5, // 변수 fruit에서 프로퍼티 이름을 동적으로 받아 옵니다. }; alert( bag.apple ); // fruit에 "apple"이 할당되었다면, 5가 출력됩니다.

in연산자로 존재여부 확인도 가능합니다

이를 활용하여 반복문제 넣을수더잇습니다

for (key in object) {}

객체는 키가 정수일 경우 정수순으로 정렬되고 아닐경우 작성된 순서대로 프로퍼티가 나옵니다
